The Dataset consists of IMDB top 1000 movies and tv shows of all the time. The original dataset can be found on Kaggle.

Data cleaning - notes

  • Removed: the keys: Certificate, Meta_score, No of votes, Gross as these values seems irrelevant related to the problem which the algorithm aim to solve.
  • The Key “Series_title” have been replaced with the key title_id - Its value is stored as a string.
  • All numbers are stored as integers or floats.
  • Value for genre and stars are respectively stored as arrays[ lists of values. ]
  • Details and people are objects within the movie object which holds a certain amount of key-value pairs, this is made by the objective to keep related data together which will make the dataset look clean and organized and makes it easier accessible through one specific key.
