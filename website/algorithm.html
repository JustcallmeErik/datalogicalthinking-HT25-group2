Description of algorithm

The purpose for the algorithm is for a user to get movie recommendation based on three input factors, genre, length (how much time the user have to watch the movie) and “actor”. The result will be sorted based on the IMDB rating, meaning the first result is highest rated.The algorithm will return a movie recommendation. There is also a loop process if the user answers yes to: “Have you seen this movie before?”

The Movie Recommender Algorithm

  1. Load the dataset.
  2. Print welcome message.
  3. Start the loop.
  4. Get user input.
    • 1 for to get a movie based on preference
    • 2 for to get a random movie recommendation
    • 3 for to exit the program
  5. If the user enter 1:
    • Get a user input for a Genre
    • Select movies for matching Genre from dataset and assign to a new dataframe.
    • Get a user input for time in minutes.
    • Add +10 and -10 mintes to the entered time.
    • Select movies for matching time range from new dataframe.
    • Get a user input for year.
    • Select movies for matching year from new dataframe.
    • Combine results from step 10 and 12 and assigned to final dataframe.
    • If the final dataframe is empty:
      • Print message ("No movie is found")
    • Else:
      • Print final dataframe
  18. If the user enter 2:
    • Randomly select and print two movies from dataset.
  20. If the user enter 3:
    • Print Goodbye message and exit the loop.
  22. Else (user enter wrong input):
    • Print an error message and go to step 3.

Limitations

  • In this algorithm, if the user enter multiple values as the input for Genre (ex: Action, Drama), program will not work. We have designed it to take only one string value (ex: Drama).
  • We ask user to enter time. We display lowest and highest duration of movies anticipating user will not enter a lower or higher number that will fall out of range of Runtime. Ideally, we should validate these input agaist limits of Runtime. However, we decided to not to make algorithm unnecessarily long.
