Information about the project

This project was made by Group 2 for the Assignment 2 - Algorithm Design (Project Work) in the Datalogical Thinking course HT25 as a part of the Masters Program in Information Science - Digital environments at the University of Bor√•s. The general idea for the assignment was to curate a dataset and design an algorithm to process the dataset in a meaningful way.

We have designed an algorithm to recommend a movie using a dataset consisting of the top 1000 movies on IMDB from 2020. The algorithm is interactive and the result is based on user input and will return a movie recommendation.

On the respective pages you can read about our motivations behind the decisions we made and get more insight into the development process. To get access to the actual algorithm you need to run it in a console and have pandas and a python debugger installed on your computer. You can read more about how to run the application in the README file in the repository.
